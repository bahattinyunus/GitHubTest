# SYLLABUS

Kısa ders planı ve hedefler:

- Hafta 1: Git ve GitHub temelleri, repo klonlama, commit, push
- Hafta 2: Python temel kavramları, küçük alıştırmalar
- Hafta 3: Test odaklı geliştirme (pytest), otomatik test bağlama
- Hafta 4: Proje teslimi ve kod incelemesi

Her alıştırma `exercises/` altında olacak ve `tests/` klasöründeki pytest testleri ile doğrulanacaktır.

Hedef: Öğrenciler temel git iş akışını, basit Python alıştırmalarını ve pytest ile test yazmayı öğrenir.
# Ders Akışı (Kısa)

1. Giriş: Git ve GitHub temelleri
2. Repo kullanımı ve alıştırma teslimi (bu repo üzerinden)
3. Branching, PR ve code review temel kavramları
4. Basit test yazma ve CI entegrasyonu
5. Öğrenci sunumları / değerlendirme

Her bölüm kısa alıştırmalar içerir. Alıştırmalar `exercises/` altında yer alır ve testlerle doğrulanır.
