01 - Giriş: Basit dosya oluşturma
================================

Amaç
- GitHub üzerinden basit bir dosya oluşturmayı ve testlerle doğrulamayı öğrenmek.

Görev
1. `exercises/01-intro/` dizinine bir dosya oluşturun: `answer.txt`
2. Bu dosyanın içeriği tam olarak şu cümle olmalı (boşluk ve büyük/küçük harf duyarlı):

   Merhaba, GitHub Eğitim!

3. Değişikliklerinizi commit ve push edin.

İpucu: Starter dosyası `starter.txt` içinde kısa bir yönerge var.
